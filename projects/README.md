# 项目代码 / Projects

本文件夹用于存放大型项目和综合练习代码。
This folder is for storing larger projects and comprehensive exercises.

## 项目结构建议 / Suggested Project Structure

```
projects/
├── final-project/
│   ├── README.md        # 项目说明
│   ├── src/            # 源代码
│   ├── tests/          # 测试代码
│   ├── docs/           # 项目文档
│   └── requirements.txt # 依赖列表
├── mini-project1/
└── ...
```

## 项目文档建议 / Project Documentation Suggestions

- 项目目标和功能说明
- 技术选型和架构设计
- 安装和运行指南
- 开发过程记录
- 遇到的挑战和解决方案
